# Customer Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using a complete data pipeline involving Python, SQL, and Power BI.

The goal is to extract meaningful insights about customer purchasing patterns, product performance, and business trends.

---

## Tools & Technologies Used

* Python (Pandas) → Data Cleaning & Preprocessing
* MySQL → Data Analysis using SQL
* Power BI → Interactive Dashboard & Visualization

---

## Key Analysis Performed

### Customer Insights

* Segmented customers into **New, Returning, and Loyal**
* Analyzed repeat buyers and subscription behavior

### Revenue Analysis

* Compared revenue between male and female customers
* Revenue contribution by different age groups

### Product Analysis

* Top 5 products based on average ratings
* Top 3 most purchased items in each category

### Business Insights

* Impact of discounts on purchasing behavior
* Shipping type vs average purchase amount
* Subscription vs non-subscription spending

---

## Dashboard

The Power BI dashboard provides:

* Interactive filters
* Category-wise product insights
* Customer segmentation visuals
* Revenue trends

---

## Key Insights

* Loyal customers form the largest segment
* Certain products show consistently higher ratings
* Discounts significantly influence purchase behavior
* Subscription does not always guarantee higher spending

---

## Conclusion

This project demonstrates end-to-end data analysis workflow — from raw data processing to business insights visualization.
